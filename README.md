# 🌊 Microwaves Project

A C++ project focused on building and simulating microwave oven operations.

---

## 🛠️ Tech Stack

- **Language:** C++
- **Compiler:** GCC / Clang / MSVC (C++11 or higher recommended)

---

## 🚀 Getting Started

### Prerequisites
- A C++ compiler installed on your system.
- Make or CMake (if applicable).

### Installation & Run

1. Clone the repository:
 ```bash
 git clone [https://github.com/carolinaml/microwaves.git](https://github.com/carolinaml/microwaves.git)
 cd microwaves
 ```

2. Compile the project:
```bash
g++ -o microwaves main.cpp
```


3. Run the executable:
```bash
./microwaves
```
